Version 2.0

This package contains 23 different animated water textures.

All Animations using 28frames in a resoulution of 512x512
and they are perfectly tileable.

To use it, just map a single animation frame to your 
water surface and add the textureAnimtion-Script to 
the object.
Select your water material in the script and
add all 28animation frames.
The speed value controls the speed of the animation.
(the higher the speed value the fast the animation).

Open the example scene to see how to use the animations.
(the textureAnimator script is applied to the waterplane
object in the example-scene).
